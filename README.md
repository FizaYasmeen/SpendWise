# 💸 SpendWise - Smart Expense Tracker

SpendWise is a full-stack expense tracking web application built with the **MERN stack**. It allows users to register, log in, and manage their daily spending via a clean and responsive interface.

---

## 🚀 Tech Stack

### Frontend
- **React** with [Vite](https://vitejs.dev/)
- **Bootstrap** for styling

### Backend
- **Node.js** + **Express.js**
- **MongoDB** (Cloud-based via MongoDB Atlas)

---


## ✨ Features

- User authentication (Register & Login)
- Dashboard for tracking expenses
- Fully responsive UI
- API integration between frontend and backend
- Clean modular codebase
